Your AI agent in iMessage

Rout is a personal AI agent that lives in your texting app. It reads your calendar, creates events, manages reminders, searches the web, and remembers context about you — all through natural conversation. No new apps. No browser tabs.

How it works

Three moving parts: a watcher, a command registry, and handlers.

Agent loop

Claude reasons through your request with multi-turn tool use. Max 5 iterations, tool results capped at 2k chars.

Safety gate

Calendar writes and reminder creation require explicit yes/no confirmation. Pending actions expire after 1 hour.

Crash-proof

Runs as a macOS launchd service. Auto-restarts after crashes and texts you if it goes down.

Extensible

Every capability is a handler: input string, output string. Add a tool in 3 steps — create, register, restart.
