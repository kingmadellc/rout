# Community Files Checklist

## Required Baseline

- `README.md` links contributor/security docs
- `CONTRIBUTING.md`
- `CODE_OF_CONDUCT.md`
- `SECURITY.md`
- `.github/ISSUE_TEMPLATE/bug_report.md`
- `.github/ISSUE_TEMPLATE/feature_request.md`
- `.github/ISSUE_TEMPLATE/config.yml`
- `.github/pull_request_template.md`

## Review Questions

- Are templates asking for reproduction and environment details?
- Are labels/titles consistent with project conventions?
- Does security policy include private reporting instructions?
- Is wording aligned with current product messaging?
