# Release Checklist

## Legal and Governance

- `LICENSE` exists and is valid
- `README.md` license section points to `LICENSE`
- `SECURITY.md` exists with private reporting channel
- `CODE_OF_CONDUCT.md` exists
- `CONTRIBUTING.md` exists

## Repository Readiness

- Issue templates and PR template exist
- CI workflow exists and passes on target commit
- Tests pass locally on target commit

## Messaging Consistency

- README, docs landing page, and issue templates match product positioning
- Outdated product names/phrases removed
